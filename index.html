관리자 로그인

로그인 안됨
이 페이지는 GitHub Pages에 index.html 하나만 올려서 서비스하는 구조입니다. Firebase는 로그인(Authentication), 데이터 저장(Firestore), 보안 규칙(Security Rules)에만 사용합니다.

주의 사항

1. 이 관리자 페이지는 관리자 계정으로 로그인한 경우에만 사용합니다.
2. GitHub Pages에 올린 뒤에도 반드시 Firebase 설정값과 Firestore 보안 규칙을 함께 맞춰야 합니다.
3. 사진/오디오 파일은 서버로 전송하지 않으며, 서버에는 텍스트 정보만 저장하는 구조입니다.

날짜별 보기는 신청시각(requestedAt)을 기준으로 필터링합니다.
상세 검색
날짜, 노트북 번호, 사용자 이름, 유형 중 입력한 조건만 검색에 사용합니다. AND는 모든 조건 일치, OR은 하나라도 일치하면 표시합니다.
월 단위 삭제는 서버 Firestore 기록만 삭제합니다. 로컬 사진/오디오는 각 PC의 30일 자동삭제 정책으로 관리됩니다.
